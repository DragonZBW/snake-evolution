Noteworthy

  1. We used Bulma for the styles and also wrote some custom CSS to get everything looking right.
  2. The Fetch API was used to get data from JSON files for the presets.
  3. All the rendering for the snake displays and neural network display is done using canvases wrapped in custom web components. Using custom components allowed the code for rendering to be packaged along with the canvas elements themselves, which helped keep things organized.
  4. The snakes learn through a standard genetic algorithm hooked up to a population of neural networks. Each snake in the population is simulated simultaneously and has a fitness score calculated over time (the user can adjust how this score is calculated). Once all the snakes in the generation have died, the fittest snakes are chosen to pass on genes to the next generation. They are copied and the neural networks get their weights and biases mutated based on the mutation rate. A variety of behaviors can be achieved just by adjusting the parameters. All the code for the neural networks was hand-written and does not use any outside libraries, although a set of tutorials from The Coding Train was used (see Resources Utilized above).
